# Analisis-Statistik-Kemiskinan-Jawa-Timur-2023
Project ini berfokus pada analisis data kemiskinan kabupaten/kota di Jawa Timur tahun 2023 menggunakan data BPS. Analisis dilakukan untuk mengetahui jumlah penduduk miskin, persentase kemiskinan, rata-rata, proporsi, dan varians data kemiskinan di setiap daerah.
